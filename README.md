# PC Budget Calculator 🖥️

Calculadora básica de presupuestos para componentes de PC. Proyecto enfocado en **Clean Code**, **TypeScript** y **Performance** con React + Vite.

## 🚀 Tecnologías
- React 18 (Functional Components)
- TypeScript (Strict Mode)
- Tailwind CSS
- Vite

## 🛠️ Instalación y Uso
1. Instala las dependencias: `npm install`
2. Corre el proyecto en local: `npm run dev`
